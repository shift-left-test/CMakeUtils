# CMakeUtils

A set of CMake macro extensions for a C/C++ project

### Requiurements

- python 2.7 or above
- pytest
- gcc/g++
- CMake 3.5.0 or above
- make

### How to install prerequisites

    $ sudo apt-get install python cmake build-essential python-pytest

### How to run the tests

    $ ./runtest.sh
